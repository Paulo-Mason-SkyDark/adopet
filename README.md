This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

First, you need install all package requirements from projet, so run:

```bash
npm install
# or
yarn install
```

## Run project

To run the development server use:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Eslint

To make all the code standardized and following good practices, the Eslint tool performs a code check to help identify errors or possible formatting problems, to run the tool use:

```bash
yarn lint
```

## Unity tests

The Jest tool is used to perform unit tests on pages and application components, to run it use:

```bash
yarn jest
```

## Important

Always run `yarn lint` and `yarn test` commands before any commit
